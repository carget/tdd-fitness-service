####Fitness service####

Сервис принимает события с количеством выполненной активности.
Например:
- 8 утра - выпил стакан воды (250 мл)
- 8:30 - позавтракал 450 Ккал
- ...

Сервис накапливает данные и может выдавать отчеты:
- сколько осталось до нормы в процентах и абсолютных величинах
- отчеты по всем активностям за периоды (в днях)
Отчеты содержат инфорацию о медиане по дням и общий прогресс выполнения активности

Сервис имеет возможность менять профайлы (наборы целевых уровней активностей).
